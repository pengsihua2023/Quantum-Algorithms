# 3. Search and Amplitude Amplification Algorithms

## 3.1 Problem setting
Search problems form one of the most intuitive settings in which quantum algorithms can outperform classical methods. The central question is whether a marked item can be found more efficiently than would be possible through classical querying or exhaustive search.

## 3.2 Grover’s algorithm
Grover’s algorithm showed that unstructured search can be accelerated quadratically in the standard oracle model. Its importance extends beyond the search problem itself, because it established a widely applicable form of quantum speedup that does not rely on highly specialized number-theoretic structure.

## 3.3 Amplitude amplification
Amplitude amplification generalizes the core idea behind Grover’s algorithm and provides a broader framework for improving the success probability of quantum procedures. This framework has had lasting influence because it turned a specific search result into a reusable algorithmic primitive.

## 3.4 Significance and applications
The significance of this family of algorithms lies not only in the search task itself but also in the general insight that quantum interference can systematically amplify desirable computational outcomes. This idea has influenced later work in optimization, estimation, and algorithmic subroutine design.

## 3.5 Limitations
Despite their theoretical importance, search-based quantum algorithms do not automatically yield overwhelming practical advantages in all settings. Their usefulness depends on how the problem is encoded, whether oracle access is realistic, and how the quantum cost compares with optimized classical heuristics in real computational environments.
